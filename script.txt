Good morning sir.

I am Prabanjan, and our project is titled “Implementation of Arthur Scherbius’s Enigma Cipher Machine.”

The main aim of our project is to simulate the encryption and decryption process of the Enigma machine using C and Python.

The Enigma machine was used by the German military during World War II to send secret coded messages.

It used rotors, plugboards, and a reflector to change every letter typed into another letter, creating a secret code.

I worked on researching the original Enigma mechanism and understanding how each part — rotor and reflector — contributed to encryption.

This study helped our team design a modern digital version of the Enigma machine.


Ruchita:

I am Ruchitha, and I worked on the encryption and decryption logic using the C programming language.

The libraries used were:

stdlib.h – for memory functions

ctype.h – for character handling

string.h – for string operations

The program takes a plaintext message and passes it through multiple logical steps, similar to the Enigma’s rotor mechanism.

It then produces an encrypted output which can only be decoded using the same settings.

We also made small modifications to the original algorithm to make it stronger and more adaptable for today’s systems.

This part of the project helped us understand how simple logic can create powerful encryption.
